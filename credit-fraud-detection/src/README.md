Credit Card Fraud Detection – End-to-End ML Project
Overview

This project is a complete machine learning pipeline for detecting fraudulent banking transactions using real-world credit card data. It includes data engineering, model training, MLOps, deployment, and monitoring, following enterprise standards used by top banks.

Raw Data → Preprocessing → Feature Engineering → Model Training → Model Artifact
      → S3 → AWS Lambda → API Gateway → Real-Time Fraud Prediction

Tech stack used :

Python, NumPy, Pandas

Scikit-learn, XGBoost

AWS S3, Lambda, API Gateway

Terraform
